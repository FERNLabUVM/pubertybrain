# pubertybrain
Project files for ABCD adversity, puberty, brain development

Participants:
The 6.0 release includes data for 5056 participants. 4086 have structural imaging data at Time 4.

Variables included in analysis (6.0 data release):

1. Participant's sex                    ab_g_stc__cohort_sex        
2. Youth's age at data collection       ab_p_demo_age
3. Adversity at baseline                ela_plus_bs                 
  (calculated following https://github.com/karalk07/abcd-ela/tree/main)
4. Girl's Puberty (youth-reported)      ph_y_pds (girls: ph_y_pds_age, ph_y_pds_001 ph_y_pds_002 ph_y_pds_003 ph_y_pds__f_001 ph_y_pds__f_002
5. Boy's Puberty (youth-reported)       ph_y_pds (boys:  ph_y_pds_age, ph_y_pds_001 ph_y_pds_002 ph_y_pds_003 ph_y_pds__m_001 ph_y_pds__m_002)
5. Gray matter volume (cortical)        See CleanBrainData.R
   Gray matter volume (subcortical)     See CleanBrainData.R
